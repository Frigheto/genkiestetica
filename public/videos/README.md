# Pasta de Vídeos

Coloque seus vídeos nesta pasta seguindo a estrutura:

## Estrutura de Pastas:
```
videos/
├── estetica/
│   ├── video1.mp4
│   ├── video2.mp4
│   └── video3.mp4
├── fisioterapia/
│   ├── video1.mp4
│   ├── video2.mp4
│   └── video3.mp4
├── massoterapia/
│   ├── video1.mp4
│   ├── video2.mp4
│   └── video3.mp4
└── pilates/
    ├── video1.mp4
    ├── video2.mp4
    └── video3.mp4
```

## Formatos Recomendados:
- **MP4** (H.264) - Melhor compatibilidade
- **WebM** - Alternativa moderna
- **Resolução:** 1280x720 (720p) ou 1920x1080 (1080p)
- **Tamanho:** Mantenha abaixo de 50MB por vídeo para melhor performance

## Como usar:
Os vídeos serão carregados automaticamente das pastas correspondentes.
